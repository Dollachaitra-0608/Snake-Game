# Snake Game 🐍

A classic Snake game built using TypeScript and HTML. The player controls a snake to eat food and grow longer, avoiding collisions with the walls or itself.

## 🔧 Tech Stack
- TypeScript
- HTML
- CSS (optional)

## 🎮 Features
- Arrow key controls
- Score tracking
- Game-over screen
- Responsive game board
